## Getting Started

#### 1. Clone the repository
```bash
git clone https://github.com/meisru/weather-dashboard.git
cd weather-app
```

#### 2. Install dependencies
```bash
npm install
```
#### 3. Create `.env` file with your `VITE_OPENWEATHER_API_KEY`

#### 4. Run the project
```bash
npm run dev
```
